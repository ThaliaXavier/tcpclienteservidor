# Como usar:
Clone o repositório:
git clone https://github.com/ThaliaXavier/tcpclienteservidor.git

Navegue até o diretório do projeto:
cd tcpclienteservidor

Execute o servidor: (em um terminal)
python servidor.py

Execute o cliente: (em outro terminal)
python cliente.py
